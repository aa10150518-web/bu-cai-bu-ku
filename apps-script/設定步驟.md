# 自動 LINE 通知與訂單統計設定

這份後台使用 Google Apps Script
功能是接收網站訂單、寫入 Google 試算表、推送 LINE 訊息給你

## 你需要準備

1. 一個 Google 試算表
2. LINE Developers 的 Messaging API Channel
3. 你的 LINE Channel access token
4. 你的 LINE User ID

## 設定 Google Apps Script

1. 開一個 Google 試算表
2. 點「擴充功能」>「Apps Script」
3. 把 `Code.gs` 內容貼上
4. 到「專案設定」>「指令碼屬性」新增三個值

```text
SPREADSHEET_ID = 你的 Google 試算表 ID
LINE_CHANNEL_ACCESS_TOKEN = 你的 LINE Channel access token
LINE_OWNER_USER_ID = 你的 LINE User ID
```

## 取得 LINE User ID

1. Apps Script 先部署成網頁應用程式
2. 把網頁應用程式網址貼到 LINE Developers 的 Webhook URL
3. 你用自己的 LINE 傳任意訊息給官方帳號
4. 官方帳號會回覆你的 User ID
5. 把那串 User ID 填到 `LINE_OWNER_USER_ID`

## 部署 Apps Script

1. 點右上角「部署」>「新增部署作業」
2. 類型選「網頁應用程式」
3. 執行身分選「我」
4. 存取權限選「所有人」
5. 部署後複製網頁應用程式網址

## 接回網站

打開 `index.html` 找到這行：

```js
const ORDER_API_URL = "";
```

把 Apps Script 網頁應用程式網址貼進去：

```js
const ORDER_API_URL = "你的 Apps Script 網頁應用程式網址";
```

重新上傳 `index.html` 後，客人送出訂單時就會：

1. 寫入 Google 試算表
2. 推送新訂單 LINE 通知給你
3. 顯示訂單完成畫面給客人
